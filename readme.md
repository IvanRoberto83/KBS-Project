## Deskripsi

Project ini merupakan implementasi Knowledge Based System (KBS) untuk mengklasifikasikan risiko gagal studi mahasiswa berdasarkan data akademik.

Metode yang digunakan:

* Rule-Based System (IF–THEN)
* Forward Chaining
* First-Hit Priority

## Kategori Risiko

* Risiko Tinggi
* Risiko Sedang
* Risiko Rendah

## Dataset

Dataset berisi data mahasiswa Informatika UKDW angkatan 2020–2021 yang telah melalui:

* Data cleaning
* Penghapusan missing values
* Penghapusan duplikasi
* Transformasi data
